## CS 340 Portfolio Reflection

### How do you write programs that are maintainable, readable, and adaptable?

I write maintainable and readable programs by organizing code into separate modules and using clear, descriptive names for variables and functions. In Project One, I created a CRUD Python module to handle all database operations. This made my code easier to manage and reuse. In Project Two, I reused this module to connect the dashboard widgets to the database, which made development faster and more efficient. In the future, this CRUD module could be reused in other applications that require database interaction, such as web apps or dashboards.

---

### How do you approach a problem as a computer scientist?

I approach problems by breaking them down into smaller parts and understanding the requirements before starting. In this project, I focused on how the database should be structured and how the dashboard would display the data. This project was different from earlier assignments because it required connecting multiple components together, including the database and the user interface. In the future, I would continue using this structured approach and spend more time planning before building.
